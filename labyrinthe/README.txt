Commandes :
carré bleu : up->z, down->s, left->q, right->d

carré rouge : up->Up, down->Down, left->Left, right->Right

Mode d'emploi :

Lancer launcher.py

Cliquer sur play

Déplacer le personnage jusqu'à la case verte


Note : Le laucher se ferme au lancement du jeu.
Si le jeu est fermé, il faut relancer le launcher.
